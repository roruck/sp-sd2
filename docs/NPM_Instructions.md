# NPM dependencies (required by SD1)

You MUST add and use:
- 1 CSS library (Bootstrap)
- 1 JS library (SweetAlert2)

In your Laravel project root run:

```bash
npm install
npm install bootstrap sweetalert2
npm run dev
```

This ZIP already includes:
- resources/css/app.css (imports Bootstrap)
- resources/js/app.js (uses SweetAlert2)
