# GitHub (very simple steps)

## Option A (recommended): GitHub Desktop
1) Create a repo on GitHub (Public): `sd1-conference-system`
2) Install GitHub Desktop and sign in
3) Clone your repo to your computer
4) Copy your **Laravel project** files into the cloned folder
5) In GitHub Desktop:
   - write a message -> Commit
   - Push origin

## Required for SD1
- Make at least **5 commits** (do small commits: routes, views, admin, translations, etc.).
